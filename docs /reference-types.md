# Типы референсов

Референсы — это визуальные якоря, которые помогают AI-модели сохранять единообразие между кадрами.

## Референсы для рекламы продукта

| Тип | Описание | Пример |
|-----|----------|--------|
| Фото продукта | Чёткое изображение продукта на нейтральном фоне | Флакон, упаковка, гаджет |
| Логотип | Векторный или PNG с прозрачным фоном | logo.png |
| Упаковка | Коробка, пакет, этикетка | Со всех ракурсов |
| Фирменные цвета | HEX или RGB значения | #D4AF37, #1A1A1A |
| Мудборд | Коллаж настроения/атмосферы | Aesthetic, стиль подачи |

## Референсы для SaaS-видео

| Тип | Описание | Пример |
|-----|----------|--------|
| Скриншот дашборда | Основной экран приложения | dashboard.png |
| Экран действия | Момент выполнения функции | calendar-create.png |
| Логотип | С прозрачным фоном | logo.svg |
| UI-элементы | Кнопки, иконки, модалки | Из дизайн-системы |
| Цвета бренда | Primary, secondary, accent | Палитра из гайдлайнов |

## Референсы для сюжетного видео

| Тип | Описание | Пример |
|-----|----------|--------|
| Персонаж | Лицо, поза, выражение | Фото модели/актёра |
| Одежда | Полный образ | Аутфит, аксессуары |
| Локация | Место действия | Интерьер, улица, студия |
| Реквизит | Повторяющиеся объекты | Чашка, книга, ноутбук |
| Освещение | Стиль света | Мягкое, контрастное, неон |

## Правила работы с референсами

- Чем выше качество исходного изображения, тем лучше результат
- Референс персонажа должен показывать лицо фронтально
- Референс продукта — на однотонном фоне, без отвлекающих деталей
- Все референсы складываются в `02-references/approved/`
- Не используйте стоковые фото с водяными знаками
