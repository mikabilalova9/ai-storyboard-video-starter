# Стартовый промпт для LLM

Скопируйте этот текст и вставьте в ChatGPT, Claude, Codex или любую другую нейросеть.

---

```
Я создаю AI-видео в стиле раскадровки.

Используй следующий рабочий процесс:
1. Креативный бриф
2. Сбор референсов
3. Список кадров (шотов)
4. Промпты для изображений
5. Фреймы раскадровки
6. Промпты для видео
7. Генерация переходных клипов
8. Склейка
9. Финальный результат

Спроси меня, какое видео я хочу создать:
- Рекламный ролик продукта
- Демо-видео для SaaS
- Видео с персонажем/сюжетом
- Кинематографичный ролик

Затем спроси, какие референсы нам нужны.

Перед началом работы спроси:
Хочешь утверждать каждый этап, или мне работать в автономном режиме
и останавливаться только если что-то рискованное или непонятное?

На каждом этапе:
- Черновики складывай в attempts/
- Утверждённые файлы — в approved/
- Отклонённые — в disapproved/
- Спрашивай разрешение перед переходом на следующий этап

Начни с помощи в написании креативного брифа.
```

---

## Дополнительные промпты

### Для продуктового ролика

```
Тип видео: рекламный ролик продукта.
Продукт: [опишите продукт].
Длительность: [15/20/30/60 секунд].
Настроение: [элегантное/динамичное/тёплое/минималистичное].
Целевая аудитория: [кто будет смотреть].
Обязательные элементы: логотип, продукт крупным планом, CTA.
```

### Для SaaS-демо

```
Тип видео: демо-видео для приложения.
Приложение: [название и назначение].
Длительность: [15/20/30 секунд].
Что показать: [ключевые экраны/функции].
Стиль: [минимализм/корпоративный/дружелюбный].
```

### Для сюжетного ролика

```
Тип видео: кинематографичный сюжетный ролик.
Сюжет: [что происходит].
Персонаж: [описание].
Локация: [где происходит].
Настроение: [драматичное/комедийное/мистическое/романтическое].
Ключевые реквизиты: [что должно повторяться в кадрах].
```
