# Новый проект

Это шаблон для нового видеопроекта.

## Как использовать

Проходите папки по порядку от 01 до 09:

1. **01-creative-brief/** — напишите бриф
2. **02-references/** — соберите референсы
3. **03-shot-list/** — разбейте на кадры
4. **04-image-prompts/** — напишите промпты для фреймов
5. **05-storyboard-frames/** — сгенерируйте изображения
6. **06-video-prompts/** — напишите промпты для видео
7. **07-transition-videos/** — сгенерируйте клипы
8. **08-stitching/** — склейте в финальное видео
9. **09-final-output/** — экспортируйте результат

## Правило

На каждом этапе:
- Черновики → `attempts/`
- Утверждённые файлы → `approved/`
- Отклонённые → `disapproved/`

Следующий шаг — только из `approved/`.
