# Начните здесь (после настройки)

Окружение настроено. Вот что делать дальше.

## 1. Изучите готовый пример

Откройте `demo-walkthrough/` и просмотрите все 9 этапов по порядку. Обратите внимание на:
- Как написан бриф
- Какие референсы собраны
- Как выглядят промпты для изображений
- Как описаны видеопереходы
- Какие проблемы были найдены при склейке

## 2. Начните свой проект

Откройте `my-first-video/` и начните с папки `01-creative-brief/`.

Скажите Claude Code или вашей нейросети:

```
Помоги написать креативный бриф для [опишите ваше видео].
Положи черновик в 01-creative-brief/attempts/brief-v1.md
```

## 3. Двигайтесь по шагам

Не перескакивайте этапы. Каждый следующий шаг строится на утверждённых результатах предыдущего.

## 4. Сохраняйте прогресс

```bash
../../tools/save-to-github.sh "Мой прогресс"
```

## Удачи!
