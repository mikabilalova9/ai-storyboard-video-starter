# AI Раскадровка: Система Создания Видео

Создавайте длинные AI-видео как режиссёр: один короткий контролируемый кадр за раз.

Вместо того чтобы просить ИИ сгенерировать одно длинное видео целиком, вы:

1. Пишете креативный бриф.
2. Собираете визуальные референсы.
3. Разбиваете видео на короткие кадры (шоты).
4. Генерируете первый и последний фрейм каждого шота.
5. Генерируете видеопереход между зафиксированными фреймами.
6. Склеиваете все клипы в финальное видео.

Это и есть весь метод.

## Визуальный процесс

Полный рабочий цикл и структура папок в одной схеме:

```
Бриф → Референсы → Список кадров → Промпты для изображений →
→ Фреймы раскадровки → Промпты для видео → Переходные клипы →
→ Склейка → Финальный результат
```

## Для кого этот репозиторий

Этот репозиторий предназначен для новичков, которые хотят получить повторяемую систему папок для AI-видео проектов.

Подходит для:

- Рекламных роликов продуктов
- Демо-видео для SaaS
- Видео с персонажами и сюжетом
- Кинематографичных роликов для соцсетей
- Лонч-видео, туториалов, рилсов

Не нужно разбираться в кинопроизводстве. Репозиторий подсказывает, что делать дальше на каждом шаге.

## Главное правило

На каждом этапе есть три папки:

```
attempts/       — черновики складываются сюда
approved/       — утверждённые файлы фиксируются здесь
disapproved/    — отклонённые файлы отправляются сюда
```

Следующий этап строится **только** из файлов в `approved/`.

Это защищает проект от хаоса.

## Скопируйте этот промпт в вашу нейросеть

После создания папки проекта вставьте этот текст в ChatGPT, Claude, Codex или любую другую LLM:

```
Я создаю AI-видео в стиле раскадровки.

Используй следующий рабочий процесс:
1. Креативный бриф
2. Сбор референсов
3. Список кадров (шотов)
4. Промпты для изображений
5. Фреймы раскадровки
6. Промпты для видео
7. Генерация переходных клипов
8. Склейка
9. Финальный результат

Спроси меня, какое видео я хочу создать:
- Рекламный ролик продукта
- Демо-видео для SaaS
- Видео с персонажем/сюжетом
- Кинематографичный ролик

Затем спроси, какие референсы нам нужны.

Перед началом работы спроси:
Хочешь утверждать каждый этап, или мне работать в автономном режиме
и останавливаться только если что-то рискованное или непонятное?

На каждом этапе:
- Черновики складывай в attempts/
- Утверждённые файлы — в approved/
- Отклонённые — в disapproved/
- Спрашивай разрешение перед переходом на следующий этап

Начни с помощи в написании креативного брифа.
```

## Пошаговый процесс для новичков

### Шаг 1: Креативный бриф

Откройте:

```
01-creative-brief/
```

Опишите, о чём будет видео.

Пример:

```
Кинематографичный рекламный ролик нового парфюма.
Флакон появляется в разных сценах: на мраморном столе,
в руке модели, на фоне ночного города.
Атмосфера — элегантность, тайна, роскошь.
```

Черновики — сюда:

```
01-creative-brief/attempts/
```

Утверждённый бриф:

```
01-creative-brief/approved/
```

Контрольный вопрос:

> Этот бриф точно описывает видео, которое вы хотите?

### Шаг 2: Референсы

Откройте:

```
02-references/
```

Добавьте всё, что модель должна воспроизводить единообразно.

Для рекламы продукта:

- Фото продукта
- Логотип
- Упаковка
- Фирменные цвета

Для SaaS-видео:

- Скриншоты приложения
- Скриншоты дашборда
- Логотип
- Фирменные цвета

Для сюжетного видео:

- Фото персонажа
- Референс одежды
- Референс локации
- Ключевые реквизиты

Контрольный вопрос:

> Это те референсы, которые мы фиксируем?

### Шаг 3: Список кадров

Откройте:

```
03-shot-list/
```

Разбейте видео на короткие шоты.

Простой вариант:

```
Шот 01: Флакон стоит на мраморном столе.
Шот 02: Рука модели берёт флакон.
Шот 03: Модель наносит аромат.
Шот 04: Флакон на фоне ночного города.
Шот 05: Финальный кадр с логотипом.
```

Улучшенный вариант с привязкой к фреймам:

```
Шот 01: Ф1 флакон на столе → Ф2 рука тянется к нему
Шот 02: Ф2 рука берёт флакон → Ф3 модель с флаконом
Шот 03: Ф3 модель с флаконом → Ф4 городской фон
```

Контрольный вопрос:

> Правильная ли это последовательность, прежде чем писать промпты?

### Шаг 4: Промпты для изображений

Откройте:

```
04-image-prompts/
```

Напишите промпты для первого и последнего фрейма каждого шота.

Пример:

```
Фрейм 01:
Крупный план элегантного стеклянного флакона парфюма на белом
мраморном столе. Мягкое боковое освещение. Золотистые блики.
На заднем плане — размытые огни вечернего города.

Фрейм 02:
Женская рука с лёгким маникюром тянется к флакону.
Тот же стол, то же освещение. Флакон в фокусе.
```

Контрольный вопрос:

> Эти промпты готовы для генерации фреймов раскадровки?

### Шаг 5: Фреймы раскадровки

Откройте:

```
05-storyboard-frames/
```

Сгенерируйте изображения по промптам.

Черновые варианты:

```
05-storyboard-frames/attempts/
```

Отобранные фреймы:

```
05-storyboard-frames/approved/frames/
```

Для каждого шота создайте папку с двумя файлами:

```
05-storyboard-frames/approved/shots/shot-01/
  first-frame.png
  last-frame.png
```

Контрольный вопрос:

> Эти фреймы достаточно единообразны, чтобы стать якорями для видео?

### Шаг 6: Промпты для видео

Откройте:

```
06-video-prompts/
```

Опишите, как должно выглядеть движение от первого фрейма к последнему.

Пример:

```
Используй first-frame.png и last-frame.png как якоря.
Длительность: 5 секунд.
Женская рука плавно берёт флакон со стола.
Камера медленно наезжает.
Сохрани тот же флакон, то же освещение, тот же маникюр.
Финальный кадр точно совпадает с last-frame.
```

Контрольный вопрос:

> Эти видеопромпты готовы к запуску?

### Шаг 7: Переходные клипы

Откройте:

```
07-transition-videos/
```

Сгенерируйте короткие клипы из утверждённых фреймов.

Черновики:

```
07-transition-videos/attempts/
```

Утверждённые клипы:

```
07-transition-videos/approved/
```

Контрольный вопрос:

> Этот клип корректно переходит от первого фрейма к последнему?

### Шаг 8: Склейка

Откройте:

```
08-stitching/
```

Соедините утверждённые клипы.

Проверьте:

- Не меняется ли персонаж/продукт между кадрами?
- Не скачет ли локация?
- Нет ли щелчков в аудио?
- Плавные ли переходы?

При необходимости используйте:

- Обрезку мёртвых кадров
- Лёгкий зум
- Белую вспышку
- Короткий кроссфейд
- Склейку на моушн-блюре

Контрольный вопрос:

> Это выглядит как одно непрерывное видео?

### Шаг 9: Финальный результат

Откройте:

```
09-final-output/
```

Поместите финальный экспорт сюда.

Также скопируйте лучший финальный файл в:

```
final-outputs/
```

Так готовые работы будет легко найти.

## Мини-пример: Рекламный ролик продукта

Цель:

```
Создать 20-секундный лонч-ролик для нового парфюма.
```

Референсы:

```
Фото флакона
Логотип бренда
Фирменные цвета
Мудборд с атмосферой
```

Список кадров:

```
Шот 01: Флакон на столе → рука берёт его
Шот 02: Модель подносит к шее → нанесение
Шот 03: Крупный план капли → блик на стекле
Шот 04: Финальный героический кадр → логотип
```

Правила единообразия:

```
Один и тот же флакон.
Один и тот же логотип.
Одна и та же цветовая палитра.
Одно и то же освещение.
```

## Мини-пример: SaaS-видео

Цель:

```
15-секундное демо-видео для приложения по планированию.
```

Референсы:

```
Скриншот дашборда
Экран календаря
Логотип
Фирменные цвета
```

Список кадров:

```
Шот 01: Хаотичный календарь → чистый дашборд
Шот 02: Клик на свободный слот → встреча создана
Шот 03: Уведомление → довольный пользователь
```

Правила единообразия:

```
Один и тот же интерфейс.
Один и тот же логотип.
Одна и та же раскладка дашборда.
Одна и та же цветовая схема.
```

```

```

Референсы:

```

Одно и то же освещение.
```

## Важные советы для начинающих

- Не генерируйте видео сразу. Сначала — фреймы раскадровки.
- Не утверждайте фреймы, которые выглядят «почти нормально». В видео станет хуже.
- Если лицо, логотип, продукт или реквизит важен — используйте референс.
- Сохраняйте отклонённые варианты. Они показывают, чего избегать.
- Короткие клипы контролировать легче, чем длинные.
- Финальное видео — это результат множества маленьких утверждений.

## Какие инструменты подходят

Этот репозиторий не привязан к конкретным инструментам.

Вы можете использовать:

- GPT-модели для генерации изображений
- Nano Banana или другие генераторы картинок
- Seedance, Kling, Runway, Pika, Veo или другие видеогенераторы
- Любую LLM для написания промптов
- `ffmpeg` для склейки

Инструмент менее важен, чем процесс.

Бриф → Референсы → Фреймы → Промпты → Переходы → Склейка → Финал.
