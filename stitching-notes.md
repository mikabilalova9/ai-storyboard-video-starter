# Заметки по склейке

Склейка — это финальная сборка видео из отдельных утверждённых клипов. Здесь важна каждая деталь.

## Частые проблемы и решения

### Видимый стык между клипами

Причина: последний фрейм одного клипа не совпадает с первым фреймом следующего.

Решения:
- Обрежьте 1–2 последних кадра первого клипа
- Добавьте кроссфейд на 0.3–0.5 секунды
- Используйте белую вспышку (0.1 сек) как маскировку
- Склеивайте на моменте быстрого движения (моушн-блюр)

### Скачок цвета

Причина: разные клипы имеют немного разную цветовую температуру.

Решения:
- Выровняйте баланс белого в каждом клипе
- Примените единый LUT/фильтр на финальное видео
- Добавьте цветокоррекцию слоем поверх всей таймлинии

### Дрифт персонажа

Причина: AI немного меняет внешность от кадра к кадру.

Решения:
- Вернитесь к этапу 5 и перегенерируйте проблемные фреймы
- Используйте более подробный промпт с указанием деталей внешности
- Прикрепите референс лица к каждому промпту

### Дрифт продукта

Причина: форма, цвет или размер продукта меняется.

Решения:
- Аналогично дрифту персонажа — фиксируйте через референс
- Укажите в промпте конкретные детали: форма, цвет, размер, расположение логотипа

### Щелчки в аудио

Причина: резкая склейка без плавного перехода.

Решения:
- Добавьте фейд-ин/фейд-аут на стыках (50 мс)
- Наложите фоновый эмбиент поверх всех клипов
- Используйте музыку, чтобы замаскировать швы

## Инструменты для склейки

- `ffmpeg` — бесплатный, мощный, работает из командной строки
- CapCut — бесплатный, простой, подходит для новичков
- DaVinci Resolve — бесплатный, профессиональный
- Adobe Premiere — платный, индустриальный стандарт

## Базовая команда ffmpeg для склейки

```bash
# Создайте файл list.txt с путями к клипам:
# file 'shot-01.mp4'
# file 'shot-02.mp4'
# file 'shot-03.mp4'

ffmpeg -f concat -safe 0 -i list.txt -c copy output.mp4
```
