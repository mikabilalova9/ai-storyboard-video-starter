# Референсы

Для этого проекта нужны следующие референсы:

## Утверждённые референсы

1. **product-reference.png** — фото флакона парфюма на белом фоне
2. **brand-logo.png** — логотип бренда с прозрачным фоном
3. **color-palette.md** — фирменные цвета

## Фирменные цвета

- Основной: #1A1A1A (глубокий чёрный)
- Акцент: #D4AF37 (золотой)
- Дополнительный: #2C2C2C (тёмно-серый)
- Текст: #F5F5F5 (мягкий белый)

## Примечание

В реальном проекте здесь лежат PNG/JPG файлы.
Для демо-примера используются описания вместо файлов.

Добавьте свои референсы в эту папку, когда будете
создавать собственный проект.
