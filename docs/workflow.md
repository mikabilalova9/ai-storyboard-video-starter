# Рабочий процесс

Рекомендуемый порядок работы от начала до конца.

## 0. Выбор режима управления

Перед началом работы задайте пользователю вопрос:

> Хочешь утверждать каждый этап вручную, или мне работать самостоятельно и останавливаться только при рисках или неясностях?

Для новичков по умолчанию: ручное утверждение каждого этапа.

## 1. Креативный бриф

Задача: определить, о чём будет видео.

Создайте:

- Замысел/сюжет
- Целевая аудитория
- Формат
- Настроение
- Главный персонаж или продукт
- Какие референсы нужны
- Что не должно меняться от кадра к кадру

Контрольный вопрос:

> Этот бриф соответствует видео, которое вы хотите?

Утверждённый файл — в:

`01-creative-brief/approved/`

## 2. Референсы

Задача: собрать визуальные якоря.

Для рекламы продукта запросите:

- Фото продукта
- Логотип
- Упаковка
- Фирменные цвета
- Нужные ракурсы продукта

Для SaaS-видео:

- Скриншоты приложения
- Логотип
- Страницы дашборда
- Состояния UI
- Стиль бренда

Для кинематографичных/сюжетных видео:

- Референс персонажа
- Референс локации
- Стилевой референс
- Реквизиты, которые должны повторяться

Контрольный вопрос:

> Это те референсы, которые мы фиксируем?

Утверждённые материалы — в:

`02-references/approved/`

## 3. Список кадров

Задача: разбить видео на короткие шоты.

Каждый шот должен содержать:

- Номер шота
- Описание первого фрейма
- Описание последнего фрейма
- Длительность
- Эмоциональный акцент
- Что меняется в кадре
- Что должно оставаться неизменным

Контрольный вопрос:

> Правильная ли это последовательность, прежде чем переходить к промптам?

Утверждённый список — в:

`03-shot-list/approved/`

## 4. Промпты для изображений

Задача: написать промпты для первого и последнего фрейма.

Видео пока НЕ генерируем.

Для каждого шота напишите:

- Промпт первого фрейма
- Промпт последнего фрейма
- Какие референсы использовать
- Правила единообразия
- Негативный промпт (чего избегать)

Контрольный вопрос:

> Эти промпты достаточно сильные для генерации фреймов раскадровки?

Утверждённые промпты — в:

`04-image-prompts/approved/`

## 5. Фреймы раскадровки

Задача: сгенерировать и утвердить визуальные якоря.

Генерируйте фреймы по порядку. Для каждого шота:

- Последний фрейм одного шота обычно становится первым фреймом следующего.
- Следите за единообразием персонажа, продукта, одежды, локации и освещения.

Контрольный вопрос:

> Эти фреймы достаточно единообразны, чтобы стать якорями для видео?

Утверждённые фреймы — в:

`05-storyboard-frames/approved/`

## 6. Промпты для видео

Задача: описать движение между зафиксированными фреймами.

Каждый видеопромпт должен включать:

- Ссылку на первый фрейм
- Ссылку на последний фрейм
- Длительность клипа
- Движение камеры
- Движение объекта
- Тайминг ключевых моментов
- Правила единообразия
- Чего избегать

Контрольный вопрос:

> Эти видеопромпты готовы к запуску?

Утверждённые промпты — в:

`06-video-prompts/approved/`

## 7. Переходные клипы

Задача: сгенерировать каждый короткий клип.

Первый и последний фреймы — как якоря. Генерируйте попытки, пока результат не устроит.

Контрольный вопрос:

> Этот клип корректно переходит от первого фрейма к последнему?

Утверждённые клипы — в:

`07-transition-videos/approved/`

## 8. Склейка

Задача: собрать утверждённые клипы в единое видео.

Проверьте:

- Стыки между фреймами
- Стыки в аудио
- Скачки цвета
- Дрифт персонажа (изменение внешности)
- Дрифт продукта (изменение формы/цвета)
- Тайминг

Используйте обрезку, зум, вспышки, стоп-кадры или кроссфейды только если они помогают скрыть стык.

Контрольный вопрос:

> Это выглядит как одно непрерывное видео?

Файлы склейки — в:

`08-stitching/approved/`

## 9. Финальный результат

Задача: экспортировать финальное видео.

Финальный экспорт — в:

`09-final-output/`

Лучший результат также скопировать в:

`/final-outputs/`

Так готовые работы будет легко найти.
